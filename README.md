# Student Management Console App / 学生管理コンソールアプリ
Java console application to manage student records with full CRUD functionality.  
Javaコンソールアプリで学生情報を管理します（CRUD機能完備）。
## Features / 機能
- **Add student / 学生を登録**  
  Enter ID, name, and age to add a new student.  
  ID、名前、年齢を入力して学生を追加します。
- **Show all students / 学生一覧を表示**  
  Display all registered students.  
  登録された全ての学生情報を表示します。
- **Update student info / 学生情報を更新**  
  Update name and age of a student by ID.  
  学生IDを指定して名前と年齢を更新します。
- **Delete student / 学生を削除**  
  Remove a student by ID.  
  学生IDを指定して学生情報を削除します。
## Usage / 使い方
1. Open terminal / ターミナルを開く  
2. Navigate to the project folder / プロジェクトフォルダに移動  
3. Compile / コンパイル:  
```bash
javac StudentManagement.java
