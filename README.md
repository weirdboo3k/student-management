# Student Management Console App / 学生管理コンソールアプリ
Java console application to manage student records with full CRUD functionality, including scores for multiple subjects.  
Javaコンソールアプリで学生情報を管理し、CRUD機能に加えて複数科目の点数管理も可能です。
## Features / 機能
 **Add student / 学生を登録**  
  Enter ID, name, age, and scores for Math, English, and Science.  
  ID、名前、年齢、数学・英語・理科の点数を入力して学生を追加します。
 **Show all students / 学生一覧を表示**  
  Display all registered students with their scores, total, and average.  
  登録された全ての学生情報と科目別の点数、合計、平均を表示します。
 **Update student info / 学生情報を更新**  
  Update name, age, and scores of a student by ID.  
  学生IDを指定して名前・年齢・点数を更新します。
 **Delete student / 学生を削除**  
  Remove a student by ID.  
  学生IDを指定して学生情報を削除します。
